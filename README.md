# ireader-bin

AUR package for [Ireader](https://github.com/IReaderorg/IReader)

</div>

## Installation

### From [`my-repo`](https://github.com/soymadip/my-repo) Repo

```bash
  curl -sL https://mydehq.github.io/my-repo/install | bash
  sudo pacman -S ireader-bin
```

### From AUR

```bash
paru -S ireader-bin
# or
yay -S ireader-bin
```

### Manually

```bash
git clone https://aur.archlinux.org/ireader-bin.git
cd ireader-bin
makepkg -si
```
